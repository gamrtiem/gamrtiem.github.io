HELLO and welcome to the ,,,,,,,.,.

starstorm 2 unoffical building guide !!!!!!

now yuo may be wondering:

"hey icebro, isnt there already a guide on the starstorm 2 github page?? "








byeah

what u need !!

  • unity hub !
  • unity version 2019.4.26f1 !
  • github desktop or the like !

now heres where things get fun !!

copy the repo into your git client of choice ! (if you have an sdd available, use it, itll make project import times much faster)



once thats done, in unity select the project and let it import !!!!!!!!!!



now you should have the project in unity, and theres gonna be a ton of errors because you havent imported the game silly !!!

there should also be a thunderkit window open, in there go to the import settings and disable legacy api (this one will cause issues) and ror2 editor kit (this one will be installed even if turned off) now go to the tab to import the game, and select your directory and import it.



once it is done importing, go to edit -> project settings -> player and under Scripting Define Symbols, put "LONGSTRINGHERE"



you will also need to update r2api.AdressableAssets, to do this go to tools -> thunderkit -> packages and search r2api. now in thunderstore plugins, go to r2api.AdressableAssets and uninstall it, then reopen the project. go back to packages and install r2api.AdressableAssets making sure the version is 1.0.3



we're not done with AdressableAssets yet though, go to packages/r2api.AdressableAssets/plugins and select the dll, makign sure to select "auto reference", then apply in the bottom right.



now, with all of that done, import the project one last time and everything should be good and war should be over and you should have 0 errors !!!!!
